# NICExtractor
NICExtractor - Extract Just Released domains from nic.ir

This project has 2 part to run,
#1 = PHP File should be uploaded on a local webserver like WAMP,XAMPP,APACHE+PHP ..
#2 = C# Programmed application run over dot net framework 4.5+

c# program get just released page on NIC.IR and save the source of page to a txt file , then PHP file goes on that source ,
finaly domain names will be extracted from source and shown on a textbox in C# APP and will copy in your clipboard to use somewhere else for batch or BULK action ..
